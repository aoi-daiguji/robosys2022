## このソフトについて
標準入力から読み込んだ数字を足し、結果を出力します。


## 必要なソフトウェア
* Python
  * テスト済み: 3.7〜3.10

## テスト環境
* Ubuntu 22.04


## インストール方法



## コマンドの入出力例
seqコマンドを利用して実行します。

> seq 5 | ./plus
